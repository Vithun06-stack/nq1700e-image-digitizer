PES is recommended for Brother machines because it stores Brother-compatible embroidery information better than DST.

DST does not store RGB thread colours. Use the thread order shown here when changing colours on the embroidery machine.

For Brother Innov-is NQ1700E, use PES as the recommended machine file. DST is available as a fallback.

Save the PES file directly to the root of the USB drive, not inside a folder.

Project JSON is for editing only. It is not a machine-ready embroidery file.
